# Layer 1 — Core Narrative & GTM Pipeline

*External-facing strategy, positioning, fundraising*

This layer contains the locked strategic thesis and all external-facing narrative documents. Everything downstream — product specs, playbooks, design principles — derives from these docs. Changes to any L1 doc trigger a review flag across all dependent layers.

## Layer Principles

1. **The strategy memo is the source of truth.** All other L1 docs derive from it. No downstream doc introduces new product decisions.
2. **Monthly review cadence (pre-PMF).** Narrative must stay current with market feedback, competitor moves, and sprint learnings.
3. **Every doc must stand alone.** A reader should get a coherent picture from any single L1 doc, with cross-references for depth.
