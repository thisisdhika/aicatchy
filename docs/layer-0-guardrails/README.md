## Layer Principles

1. **Compliance before revenue.** Legal and regulatory requirements are non-negotiable and block any dependent workstream.
2. **Quarterly review cadence.** Guardrails are reviewed every quarter or on any triggering event (funding, regulatory change, compliance incident).
3. **Steward owns, Pathfinder reviews.** Financial and legal decisions have a single DRI with a second set of eyes.
